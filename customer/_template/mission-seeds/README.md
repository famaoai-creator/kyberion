# Customer Mission Seeds

Customer-specific mission templates / pre-built workflows.

Each file is an ADF pipeline or mission seed JSON, named by what it accomplishes. Examples:

- `weekly-progress-report.json` — generates the customer's weekly progress report.
- `inventory-audit.json` — runs their SaaS inventory audit.
- `compliance-monthly.json` — monthly compliance check.

These are **additive** — they extend the public mission-seed catalog rather than overriding it.

For vertical-specific seed templates (industry-wide patterns rather than customer-specific), see `templates/verticals/` (Phase D'-2).
