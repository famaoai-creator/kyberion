# Intelligence: Compounding Agent Value (Bounty-Training Model)

## 1. 概要 (Overview)
AIエージェントの経済活動において、短期的な報酬（トークン、法定通貨、カルマ）の獲得は、線形的な成長に留まる。本知見は、バウンティ（報奨金タスク）を「有償のトレーニングデータ」と定義し、得られた論理を永続的なスキルへと変換する複利成長モデルを提案する。

## 2. コア・メカニズム
- **Skill Extraction**: タスク解決に使用された一時的なスクリプトや正規表現、APIワークフローを `Wisdom Distiller` でスキャンし、`libs/` または `skills/` への恒久的な登録を試みる。
- **Memory as Asset**: 実行ログを単なる「日記」として捨てず、将来の類似タスクにおける「自己教師あり学習」の素材として版管理（Git Checkpointing）する。
- **Outcome**: 1つの仕事を完了するたびに、エージェントは「資金」だけでなく「新たな能力」を獲得し、次回のタスク解決コストを低減させる。

## 3. 指針
エージェントはバウンティを選択する際、報酬額だけでなく「その仕事を通じてどのような新規スキルが蒸留可能か」を優先順位の判断基準（ROI）に含めるべきである。
