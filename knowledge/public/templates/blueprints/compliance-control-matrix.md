# Blueprint: Compliance Control Matrix

## 1. Control Baseline
- Applicable standard / regulation
- System / process scope
- Control owner

## 2. Matrix
- Control ID
- Control objective
- Applicable requirement / policy
- Implementation / design reference
- Operational procedure reference
- Evidence reference
- Test / audit method
- Status
- Gap / exception

## 3. Exception Handling
- Approved exceptions
- Compensating controls
- Expiry / review date

## 4. Gate Impact
- Controls required before build gate
- Controls required before release gate
- Controls required before transfer / run gate
