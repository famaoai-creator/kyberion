# Blueprint: Deployment Verification Report

## 1. Deployment Summary
- Release / build identifier
- Environment
- Deployment window
- Deployment operator

## 2. Verification Scope
- Functional smoke checks
- Integration checks
- Data migration checks
- Monitoring / alerting checks

## 3. Results
- Check item
- Expected outcome
- Actual outcome
- Status
- Evidence reference

## 4. Issues Found
- Issue summary
- Severity
- Workaround / mitigation
- Owner

## 5. Readiness Decision
- Service stable / unstable
- Rollback required or not
- Approval / sign-off
