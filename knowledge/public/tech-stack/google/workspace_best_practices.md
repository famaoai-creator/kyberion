# Google Workspace 連携ベストプラクティス

## 1. データの可視化 (Sheets)

- 財務シミュレーション結果は、グラフ化しやすいように固定のセル範囲に出力する。
- 条件付き書式を用いて、予算超過（赤）や目標達成（緑）を強調する。

## 2. コラボレーション (Docs)

- 納品物のドラフトは Google Docs の「提案モード」で共有し、人間側の最終修正を促す。

## 3. セキュリティ

- 共有設定は常に「特定のドメイン内のユーザー」に制限し、パブリック共有を避ける。
