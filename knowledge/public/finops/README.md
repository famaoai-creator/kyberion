# finops/
CEO / CFO 監修：AIビジネス価値の最大化

## 1. FinOps Principles (FinOps Foundation)
- **Inform, Optimize, Operate**: コストを可視化（L1）し、トークン消費を最適化し、自律的に運用する 3 段階のサイクル.
- **Unit Economics**: 1 ミッションあたりの提供価値（ROI）を算出し、コスト効率を計測する.
- **Shared Responsibility**: AI エージェント自身が、実行前に自らの見積もりコスト（Token Estimate）を計算・報告する義務.

## 2. Waste Detection
- **Orphaned Contexts**: 未使用のコンテキスト保持による「トークン垂れ流し」の検知.
- **Model Right-sizing**: タスクの複雑度に対して、過剰な高性能モデルを使用していないかの常時監視.
