# Audience Strategy for Executive Reporting

対象読者の「関心事」と「痛み」に最適化されたプレゼンテーション戦略。

## 1. Executive / Board Layer (経営層)
- **関心事**: ROI（投資対効果）、リスクの金額換算、納期、競合優位性。
- **ストーリー構成**: 
    1. 結論（Go/No-Go）
    2. ビジネスインパクト（コスト削減額、売上貢献）
    3. リスク緩和策（ACEの結果）
- **禁止事項**: 詳細なソースコード、複雑なシーケンス図、技術用語の羅列。

## 2. Technical / PM Layer (現場・管理職)
- **関心事**: 開発効率、技術負債の解消状況、リソースの負荷、依存関係の解消。
- **ストーリー構成**:
    1. 現在のステータス（Active Missions）
    2. 解決済みのボトルネック
    3. 次の課題と必要なリソース
- **推奨事項**: `ux-visualizer` による最新構成図、`PERFORMANCE_DASHBOARD` の推移。

## 3. Operations / SRE Layer (運用チーム)
- **関心事**: 安定性、SLO遵守率、有事の対応手順（BCP/DR）、保守性。
- **ストーリー構成**:
    1. 稼働実績と異常検知ログ
    2. 自己修復（Self-healing）の実行エビデンス
    3. 今後の監視強化ポイント
