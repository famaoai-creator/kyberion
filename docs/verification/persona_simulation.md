# 初心者向け定型シナリオ シミュレーションレポート

本ドキュメントでは、「生成AIで業務を自動化したい初心者（非エンジニア）」向けのADFパイプラインが提供するユーザー体験をシミュレーションします。

## 1. コンセプト
初心者が複雑なプロンプトや複数ツールの連携を意識することなく、**「トピック名」を1つ入力するだけ**で、情報収集からメディア生成、資料作成までを完了させる体験を目指しています。

---

## 2. 実装されたパイプラインのシミュレーション

### シナリオA：競合調査スライド自動作成
**使用パイプライン**: `persona-beginner-competitor-research.json`

| フェーズ | アクション / 体験 |
| :--- | :--- |
| **パラメータ入力** | ユーザーが `target_name` に「競合他社A」と入力。 |
| **自動処理 (裏側)** | 1. `browser-actuator` が Google 検索を実行。<br>2. `wisdom-actuator` が検索結果から特徴・強みを抽出。<br>3. `media-actuator` が抽出データに基づきスライドを構成。 |
| **成果物確認** | `active/shared/exports/競合他社A_調査レポート.pptx` が生成される。<br>ユーザーはファイルを開くだけで、整理された比較資料を確認できる。 |

### シナリオB：新商品企画ビジュアライザー
**使用パイプライン**: `persona-beginner-product-concept.json`

| フェーズ | アクション / 体験 |
| :--- | :--- |
| **パラメータ入力** | ユーザーが `topic` に「AIペットロボット」と入力。 |
| **自動処理 (裏側)** | 1. `wisdom-actuator` が企画の骨子（コピー、機能）を作成。<br>2. `media-generation-actuator` がコンセプト画像とBGMを生成。<br>3. `media-actuator` がそれらを統合した企画書PPTを出力。 |
| **成果物確認** | PPT、画像、BGMの3点が揃う。<br>「言葉だけでここまで形になる」という感動体験を提供。 |

### シナリオC：最新ニュースまとめスライド
**使用パイプライン**: `persona-beginner-news-summary.json`

| フェーズ | アクション / 体験 |
| :--- | :--- |
| **パラメータ入力** | ユーザーが `keyword` に「脱炭素技術」と入力。 |
| **自動処理 (裏側)** | 1. `browser-actuator` が Google ニュースを検索。<br>2. `wisdom-actuator` が最新の重要トピックを3つ選別し要約。<br>3. `media-actuator` が要約をスライド化。 |
| **成果物確認** | `active/shared/exports/脱炭素技術_ニュースまとめ.pptx` が生成される。<br>毎朝の状況把握が1クリックで完了する。 |

---

## 3. 初心者にとっての価値
- **思考の外部化**: 曖昧な「〜について知りたい」「〜の企画を立てたい」という意図を、具体的な成果物（PPT、画像、音楽）に変換します。
- **摩擦の解消**: 検索、ブラウジング、要約、コピペ、PPT編集といった細かい作業の連鎖を Kyberion が肩代わりします。
- **マルチモーダルな満足度**: テキストだけでなく、視覚（画像）と聴覚（BGM）を組み合わせることで、非エンジニア層が「AIの凄さ」を直感的に感じられるようにしています。

## 4. 今後の展望
- **音声入力への対応**: `voice-actuator` と連携し、声でのトピック入力から即座に生成を開始する「ハンズフリー自動化」への拡張。
- **フィードバックループ**: 生成されたPPTに対する「もう少し可愛くして」「価格情報を詳しく」といった追加の話し言葉による微修正機能。
